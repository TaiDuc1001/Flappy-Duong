# Flappy-Duong
> Remake flappy bird and replace the bird by Duong
## Usage:
### 1. Move your directory into FlappyDuong/
```
cd FlappyDuong/
```
### 2. Install packages by running this in your CLI:
```
pip install -r .\requirements.txt
```
### 3. Run main.py
```
python main.py
```
